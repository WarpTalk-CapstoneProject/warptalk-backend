# Data Model: Transcript Service

## 1. Transcript
Tracks the metadata for a transcript session within a translation room.

| Field | Type | Modifiers | Description |
|-------|------|-----------|-------------|
| `id` | UUID | PK | Unique transcript ID |
| `translation_room_id` | UUID | Index | Links to the room |
| `status` | ENUM | NOT NULL | `ACTIVE`, `PAUSED`, `COMPLETED` |
| `total_segments` | INT | Default 0 | Counter for persisted segments |
| `created_at` | TIMESTAMP | NOT NULL | UTC |
| `updated_at` | TIMESTAMP | NOT NULL | UTC |

## 2. TranscriptSegment
The core spoken utterance generated by STT.

| Field | Type | Modifiers | Description |
|-------|------|-----------|-------------|
| `id` | UUID | PK | Segment ID (from AI/Redis) |
| `transcript_id` | UUID | FK, Index | Parent transcript |
| `participant_id` | UUID | Index | The speaker |
| `source_language` | VARCHAR(10)| NOT NULL | e.g. `en-US` |
| `original_text` | TEXT | NOT NULL | Raw transcribed text |
| `start_time` | BIGINT | NOT NULL | Timestamp offset |
| `end_time` | BIGINT | NOT NULL | Timestamp offset |
| `is_final` | BOOLEAN | Default true| Indicates if partial or final |
| `created_at` | TIMESTAMP | NOT NULL | UTC |

## 3. TranscriptTranslation
Translations of a specific segment into target languages.

| Field | Type | Modifiers | Description |
|-------|------|-----------|-------------|
| `id` | UUID | PK | Unique translation ID |
| `segment_id` | UUID | FK, Index | Parent segment |
| `target_language` | VARCHAR(10)| NOT NULL | e.g. `vi-VN` |
| `translated_text`| TEXT | NOT NULL | The translated content |
| `created_at` | TIMESTAMP | NOT NULL | UTC |

*Constraint*: `UNIQUE(segment_id, target_language)`

## 4. TranscriptCorrection
User-submitted corrections to segments or translations to improve future AI accuracy.

| Field | Type | Modifiers | Description |
|-------|------|-----------|-------------|
| `id` | UUID | PK | Correction ID |
| `segment_id` | UUID | FK | The segment being corrected |
| `participant_id` | UUID | Index | User making correction |
| `correction_type`| ENUM | NOT NULL | `ORIGINAL`, `TRANSLATION` |
| `target_language`| VARCHAR(10)| NULL | Null if type is `ORIGINAL` |
| `corrected_text` | TEXT | NOT NULL | The new text |
| `created_at` | TIMESTAMP | NOT NULL | UTC |

## 5. Glossary / GlossaryTerm
Custom vocabulary to enforce specific translations.

**Glossary**:
- `id` (UUID), `transcript_id` (UUID) or workspace-level, `name` (VARCHAR)

**GlossaryTerm**:
- `id` (UUID), `glossary_id` (UUID), `source_term` (VARCHAR), `target_term` (VARCHAR), `target_language` (VARCHAR)
